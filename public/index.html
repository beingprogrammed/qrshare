How does it work?

This web app works completely locally and offline. It doesn't upload your file to any server.

  1. Encode: The sending device compresses the file and breaks it down into small chunks (200-300 bytes each). It wraps each chunk in a header containing the file details, sequence index, and checksum.
  2. Animate: It generates a sequence of QR codes, which it displays sequentially in a loop at high speeds (e.g., 10 frames per second).
  3. Scan: The receiving device uses its camera to scan these QR codes in any order. The visual sync indicator flashes on each frame transition to notify the camera when a new chunk is active.
  4. Assemble: The receiving browser populates the chunk grid, verifying the checksum for each piece. Once all chunks (including the metadata chunk) are collected, it merges the data, decompresses it (gzip), and initiates a download.
